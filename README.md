# Church Slavonic fonts

This package contains Unicode Church Slavonic fonts. Following fonts are included:

* PonomarUnicode
* FedorovskUnicode
* IndictionUnicode
* MenaionUnicode
* MonomakhUnicode
* PomorskyUnicode
